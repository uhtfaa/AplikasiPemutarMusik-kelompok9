# AplikasiPemutarMusik-kelompok9

Struktur data yang digunakan yaitu Doubly Linked List dan Queue. Fitur dibagi menjadi 2, fitur untuk admin dan fitur untuk user.

a. Fitur Admin
   - Menambahkan lagu baru ke dalam library.
   - Melihat semua lagu yang ada di dalam library.
   - Mengubah/update data lagu yang ada di dalam library, admin dapat memilih mana yang akan di update.
   - Menghapus data lagu yang ada di dalam library.

b. Fitur User
   - Mencari lagu berdasarkan data lagu (id atau judul)
   - Memutar lagu dari library. User dapat next atau prev atau stop saat lagu di putar dan akan memutar lagu yang mirip (artis/genre) dengan lagu sebelumnya.
   - Membuat dan mengatur playlist, seperti memasukkan lagu dari library ke dalam playlist, menghapus lagu dari playlist, putar lagu dari playlist, melihat isi playlist, dan         hapus playlist yang telah dibuat.
   - Mengelola antrian lagu di playlist. User dapat membuat antrian dari lagu-lagu yang ada di playlist, melihat isi antrian dan memutar lagu dari antrian, user tidak dapat          next/prev/stop saat diputar.
   - Like dan unlike lagu yang ada di library. Melalui menu Lagu Favorit, user dapat menyukai, unlike dan melihat lagu yang di like.
